## Circuit Setup:
Create a circuit like you did in the previous assignment and write a python script to make it blink exactly once for one second.

## Using CRONTAB:
Watch the video below to learn how to use CRONTAB to run your light on startup.

[![How to Run a Script on Startup for Raspberry Pi](https://img.youtube.com/vi/Gl9HS7-H0mI/0.jpg)](https://www.youtube.com/watch?v=Gl9HS7-H0mI)

Modify your CRONTAB file to turn your light on at a specific time.  Your light should turn on later today at exactly 12:55 and the number of seconds equal to the number of your raspberry pi.  For example, if you have #8, your light should blink at 12:55:08 pm today.  We will line them all up in order and see if we can all make this work today.
